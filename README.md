# TSVFile

## 簡介

TSVFile 是一個使用 C# WinForms 開發的英文字彙學習工具。讓學習單字變得更直覺與互動化。

---

## 功能介紹

### 1. 載入單字檔

點選：
檔案 → 開啟

可選擇：
- .tsv
- .txt

載入後會將單字顯示在 ListView 中。

---

### 2. 即時搜尋單字

在搜尋框輸入英文單字前綴，例如：

ab

系統會即時篩選符合條件的單字（前綴搜尋）。

- 支援即時更新
- 清空搜尋會回復全部單字
- 顯示搜尋結果數量

---

### 3. 單字發音播放

在單字列表中：

雙擊任一單字即可播放對應音檔。

---
### 4. 鍵盤快速操作

本程式支援鍵盤快捷鍵操作，提高操作效率：

#### 檔案選單
- **Alt + F** → 開啟檔案選單
- **Alt + O** → 開啟單字檔
- **Alt + X** → 離開程式

離開程式時會跳出確認視窗：
- 按 **Y** → 確定離開
- 按 **N** → 取消離開

---

#### 幫助與資訊
- **Alt + H** → 開啟說明
- **Alt + A** → 開啟關於視窗

---


## 新增功能說明

### 搜尋功能

- 即時搜尋
- 忽略大小寫
- 自動更新 ListView

---

### 播放音效功能

- 雙擊欲學習之單字即可聽取發音

---

## 使用說明

1. 開啟程式
2. 點選「開啟檔案」
3. 載入 TSV/TXT 單字檔
4. 使用搜尋框查找單字
5. 雙擊單字播放發音
